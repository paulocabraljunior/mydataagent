# Google Ads BI Agent

This project contains a multi-agent system for analyzing Google Ads data.

## Structure

- `agents/`: Contains the agent implementations.
- `mcp/`: Model Context Protocol implementation for Google Ads.
- `a2a/`: Agent-to-Agent communication layer.
- `config/`: Configuration files.
- `tests/`: Unit and integration tests.
- `examples/`: Sample data and outputs.
